Shaky Animation using css

Another semi-useless experiment

from @tylergaw Lab

I'm not sure if it has a name, [Darren let me know that the name is Squigglevision] but I've always loved the shaky animation style of a lot of cartoons. The epic Rejected is a good example. Here I'm using a CSS animation technique i've used before to interpolate between frames of text that are slightly different to accomplish that shaky style in the browser. The images are svg so they should be nice and crisp at any size or ppi.
